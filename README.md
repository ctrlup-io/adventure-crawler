## Usage

1. Use Node.js required version:

```
nvm use
```

2. Install dependencies:

```
npm i
```

3. Set environment variables:

```
cp .env.example .env
```

4. Start proxy:

```
npm run proxy
```

5. Start app in development mode:

```
npm run dev
```
